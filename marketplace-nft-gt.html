◈ Guía para Canva

  • Tamaño recomendado: 1080 x 1080 px (cuadrado) o 1080 x 1350 px (portrait) para Marketplace
  • Fondo: Negro puro #000000 o degradado oscuro #020010 → #000820
  • Tipografía principal: Orbitron (disponible en Canva) para títulos
  • Tipografía secundaria: Exo 2 o Rajdhani para texto descriptivo
  • Paleta: Cyan #00F5FF / Dorado #FFAA00 / Magenta #FF00AA / Verde #00FF66
  • Agregar efecto glow/brillo en los íconos usando sombra exterior del mismo color
  • Incluir siempre: Bandera 🇬🇹, ID del servicio (#WEB-001), y precio en Quetzales
